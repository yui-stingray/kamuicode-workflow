The generation process is divided into two stages for optimal quality. First, a high-resolution, photorealistic still image of the near-future Tokyo with the orbital elevator is created using an image generation model (Imagen4 Fast). This focuses on achieving maximum detail, realistic lighting, and a compelling composition. Second, this static image is used as the basis for a video generation model (Hailuo-02 Pro) to create a dynamic, 15-second aerial shot with complex camera movements like spiraling and whirling, ensuring visual consistency.
